# Tkinter Music Player

A simple desktop **MP3 music player** built with **Python, Tkinter, and Pygame**.  
You can play songs, create playlists, shuffle music, track your most-played songs, and resume playback from where you left off.


## Features

- Play, pause, stop, next, and previous  
- Create and manage playlists  
- Shuffle playlist  
- Song progress slider  
- Saves playlists and playback state
